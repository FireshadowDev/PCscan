# PCscan
Aplicación para mostrar objetos en realidad aumentada mediante la cámara del móvil al detectar un marcador. El objeto mostrado tiene el propósito de permitir al usuario ver al mismo en un entorno real, respetando sus dimensiones. La interacción con el mismo dirige al usuario a la web para comprarlo.

## Instalación y Uso
1.  Descarga el archivo `pcscan.apk` en tu dispositivo Android.
2.  Apunta la cámara hacia el **marcador**.
3.  Interactúa con el objeto 3D para acceder a la plataforma de compra.
